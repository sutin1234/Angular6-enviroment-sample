Angular Environment Variables Demo

EnvService contains the following values:

{{ env | json }}

You can use env.js to control your environment variables without the need for a rebuild, as explained in How to use environment variables to configure your Angular application without a rebuild.

When you build your app using ng build --prod, you can now either:

  • Update the environment values in env.js to configure your Angular application without rebuilding it.
  • Remove env.js in the dist directory to fall back to the default values in EnvService.
